# Important Objects

This document records significant non-character objects in the Jungle world.

Named living beings (such as the royal horses) are documented as characters rather than objects.

---

# obj_hippu_coins

## Name

Hippu Coins

## Type

Currency

## Description

The standard currency used throughout the known Jungle kingdoms.

Taxes are primarily collected in Hippu Coins.

Kings may distribute Hippu Coins to ministers and others as payment or reward.

## Current Status

In circulation.

---

# obj_ancient_hippu_wisdom_books

## Name

Ancient Hippu Wisdom Books

## Type

Books

## Description

A collection of ancient writings studied by Hippu King.

They preserve Hippu philosophy, wisdom, leadership, and historical knowledge.

Hippu King regularly reads these books before sleeping.

## Current Owner

Hippu King

## Current Location

Hippu Palace

# obj_hidden_treasure

## Name

Hidden Ancestral Treasure

## Type

Treasure

## Description

A treasure hidden long ago by Bhaiya Ji's ancestors.

Whether Hathiya will ever reveal the location remains unknown.
